# Noita Entangled Worlds Seed Tool

Noita の Entangled Worlds (EW) 環境下で、World Seed と SteamID64 を入力するとパーク（聖なる山）の選択結果を表示する静的 Web アプリ。

## 使い方

1. `index.html` をブラウザで開く（またはシンプルな HTTP サーバで配信）
2. World Seed（uint32）と SteamID64 を入力
3. 各 Mountain で選ばれるパーク一覧が表示される
