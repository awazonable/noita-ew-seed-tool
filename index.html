EW Perk Viewer

Noita Entangled Worlds · per-player perk deck simulator
Debug: seed calculation
Note: This simulator replicates the EW randomize_perks feature. EW overrides the per-player seed via SetRandomSeed(1+sx, 2+sy) and delegates to vanilla perk_get_spawn_order, which uses Noita's dual-LCG PRNG (214013×x+2531011 XOR 17405×y+10395331, 15-bit output) with draw-without-replacement. Requires randomize_perks = true in EW proxy settings (default: on).
